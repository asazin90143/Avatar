# Avatar: Elemento (MVP)

A retro, turn-based fighting game based on "Avatar: The Last Airbender".

## Features
- **5 Classes:** Water, Fire, Earth, Air, and Avatar.
- **Power Scaling:** Rock-Paper-Scissors damage logic (e.g., Water beats Fire).
- **Unique Skills:** Healing, Burning, Stun, and Evasion.
- **Avatar State:** The Avatar character can switch elements mid-battle.

## How to Run
1. Download `index.html`, `style.css`, and `game.js`.
2. Place them all in the same folder.
3. Double-click `index.html` to open it in your web browser. 
   *(Chrome, Firefox, Safari recommended)*.

## Controls
- Use your mouse to select characters and click attack buttons.
- If playing as **Avatar**, use the colored buttons above the attack menu to switch elements.

## Customization
To add your own pixel art:
1. Save your images as `.png` files (e.g., `fire-bender.png`).
2. Open `style.css`.
3. Find `.fire-bender` and add `background-image: url('fire-bender.png');`.
